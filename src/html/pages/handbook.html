🧩 HTML Компоненты (Partials)

Header

@@include('header.html')

Шапка сайта. Подключается в начале каждой страницы.

Footer

@@include('footer.html')

Подвал сайта. Подключается в конце каждой страницы.

📜 JavaScript Модули

main.js

src/js/main.js

Основной файл инициализации. Точка входа для всех скриптов.

mobile-menu.js

src/js/mobile-menu.js

Логика мобильного меню (открытие/закрытие, переключение классов).

modal.js

src/js/modal.js

Управление модальными окнами. Интеграция с @fancyapps/ui.

⚙️ NPM Команды

Команда Описание Режим
npm run dev Запуск сервера разработки с автообновлением dev
npm run build Сборка проекта в production (без минификации) prod
npm run clean Очистка папки dist/ clean
npm run lint Проверка кода через ESLint dev
npm run lint:fix Исправление ошибок ESLint автоматически dev
npm run format Форматирование кода через Prettier dev
npm run format:check Проверка форматирования без изменений dev

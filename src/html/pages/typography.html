1. Базовая статья (prose)

Введение в Tailwind CSS

Tailwind CSS — это утилитарный CSS-фреймворк, который позволяет быстро создавать современные интерфейсы без написания собственного CSS. В отличие от таких библиотек, как Bootstrap, Tailwind не предоставляет готовых компонентов.

Почему Tailwind?

Вот несколько причин использовать Tailwind CSS:

  • Быстрая разработка интерфейсов
  • Минимальный размер итогового CSS
  • Полная кастомизация через конфигурацию
  • Отличная документация и сообщество

Пример кода

<button class="bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded">
  Click me
</button>

Tailwind CSS позволяет сосредоточиться на создании интерфейса, а не на придумывании имён классов.

Для установки Tailwind CSS v4 выполните команду: 

npm install tailwindcss @tailwindcss/postcss

2. Кастомные стили (prose-blue)

Синяя тема

Класс prose-blue применяет синюю цветовую схему к заголовкам и ссылкам. Это полезно для согласования с брендом вашего проекта.

Пример ссылки будет синего цвета вместо стандартного.

Цитата также будет оформлена в синих тонах.

3. Тёмная тема (prose-invert)

Инвертированная тема

Класс prose-invert предназначен для тёмных фонов. Он инвертирует цвета текста и заголовков, делая их светлыми.

  • Белый текст на тёмном фоне
  • Светло-серые заголовки
  • Контрастные ссылки
<div class="bg-gray-900">
  <article class="prose prose-invert">
    <!-- Контент -->
  </article>
</div>

4. Размеры типографики

prose-sm (компактный)

Маленький размер

Этот текст имеет уменьшенный размер шрифта. Подходит для боковых панелей, карточек и других компактных элементов интерфейса.

prose-lg (крупный)

Большой размер

Увеличенный размер шрифта для лучшей читаемости. Идеально подходит для блогов, статей и длинных текстов.

prose-xl (очень крупный)

Очень большой размер

Максимальный размер для особого акцента. Используйте для важных объявлений или специальных страниц.

5. Полная статья в блоге

Как создать блог на Tailwind CSS

Пошаговое руководство по созданию современного блога с использованием Tailwind CSS v4 и плагина Typography.

Шаг 1: Установка

Начните с установки Tailwind CSS и необходимых зависимостей:

npm install tailwindcss @tailwindcss/postcss @tailwindcss/typography

Шаг 2: Настройка

Добавьте плагин Typography в ваш CSS файл:

@import 'tailwindcss';
@plugin "@tailwindcss/typography";

Шаг 3: Создание контента

Используйте класс prose для стилизации статей:

<article class="prose prose-lg max-w-none">
  <h1>Заголовок статьи</h1>
  <p>Текст статьи...</p>
</article>

Советы по использованию

  1. Используйте prose-lg для длинных статей
  2. Добавляйте prose-blue для цветных акцентов
  3. Применяйте prose-invert для тёмных тем
  4. Комбинируйте с max-w-none для полной ширины

Важно: Плагин Typography автоматически стилизует все вложенные элементы, так что вам не нужно добавлять классы к каждому тегу.

Заключение

Tailwind CSS Typography — это мощный инструмент для создания красивых текстовых страниц. Он экономит время и обеспечивает согласованный внешний вид всего контента.
